# Support

## How to Get Help

This project uses GitHub issues and discussions for support.

### Before You Ask

1. **Read the [README](README.md)** — it covers setup, configuration, and
   common usage.
2. **Search [existing issues](../../issues)** — someone may have already
   reported or solved your problem.
3. **Check [discussions](../../discussions)** — community Q&A may have your
   answer.

### Where to Ask

| Channel                                  | Use For                                      |
| ---------------------------------------- | -------------------------------------------- |
| [Issues](../../issues)                   | Bug reports, reproducible problems            |
| [Discussions](../../discussions)         | Questions, ideas, general help                |
| [Security Advisories](../../security)    | Security vulnerabilities (see [SECURITY.md](SECURITY.md)) |

### Tips for a Good Question

- **Be specific**: Include the mode you're using, your Azure region, and
  relevant config.
- **Include logs**: Attach browser console output or server logs.
- **Show your setup**: Share your Python/Node version, OS, and relevant
  environment variables (redact secrets).
- **Minimal reproduction**: If possible, provide a minimal set of steps to
  reproduce the issue.

## Azure-Specific Support

For issues related to Azure services (Azure OpenAI, Azure AI Foundry), consult:

- [Azure OpenAI documentation](https://learn.microsoft.com/azure/ai-services/openai/)
- [Azure support plans](https://azure.microsoft.com/support/plans/)
